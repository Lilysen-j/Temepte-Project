i made this project as a temeplte for others to base their godot games on.

included:
* hitbox
* hurtbox
* healthnode
* playermovement node

# how to install
get git locally by doing 
```bash git pull https://github.com/Lilysen-j/Temepte-Project/```

then open godot and press **create new project**

navigate to where you put this repository and select the folder

select **ok**
